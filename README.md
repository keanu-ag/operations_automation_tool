# Marketplace Optimization & AI Insights

### Proyecto de Automatización y Analítica Avanzada
Este prototipo resuelve el problema de asignación de pedidos en un marketplace de tres partes, optimizando la logística de última milla.

## Stack Tecnológico
- **Optimization**: Programación lineal con `PuLP` (Minimización de distancia).
- **Backend**: Python & Pandas.
- **Database**: Supabase (PostgreSQL) para persistencia de logs operativos.
- **AI**: Gemini para generación de insights estratégicos.
- **Frontend**: Streamlit + Plotly para visualización geoespacial interactiva.

## 📈 Impacto de Negocio
- **Reducción de tiempo de llegada**: Al asignar al repartidor más cercano.
- **Eficiencia de targa**: Restricciones dinámicas para evitar saturación de socios.
- **Decisiones basadas en datos**: Transformación de métricas técnicas en recomendaciones ejecutivas vía LLM.
